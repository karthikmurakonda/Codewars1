# Natural Sum
## Statement
Find the sum of odd integers from 1 to N.
## Input
First line : T (number of test cases)
Second line : N (integer till what the sum is to be calculated)
## Constraints 
1<=T<=5

1<=N<=10<sup>6</sup>
## Sample input 
```
1
11
```
## Sample output
```
36
```
## Explanation
For the given test case , we have odd integers **1, 3, 5, 7, 9, 11** and the sum is **1+3+5+7+9+11 = 36** .
